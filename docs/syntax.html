Basic Syntax

Here are some of the basic syntax elements of the Rune language:

// Variable declaration
x = 10

# Function declaration
add = fun(a, b) {
    a + b # the last expression will be returned
}

# Conditional statement
if x > 0 {
    print("x is positive")
} else {
    print("x is non-positive")
}

// Loop
while i > 10 {
    println(i)
    i = i + 1
}
